This material is part of the exercises for the **The Analytics Edge (edX/MITx)** course on edX and has been adapted to run with **Apache Spark (PySpark DataFrame API)**.

##Structure

- `AnalyticsEdge_AnAnalyticalDetective_Questions.ipynb`
- Questions and spaces to solve.

- `AnalyticsEdge_AnAnalyticalDetective_Solutions.ipynb`
- Final answers (without code).

- `AnalyticsEdge_AnAnalyticalDetective_PySpark.ipynb`
- Complete implementation in PySpark (executable code).

##Requirements

- Python 3.9+
- PySpark (Spark 3.x recommended)
- Jupyter Lab or Jupyter Notebook
- (Optional) pandas and matplotlib for some graphs

## Dataset

Place the `mvtWeek1.csv` file in the same directory as the notebooks (or modify `DATA_PATH`).
